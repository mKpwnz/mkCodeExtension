# Upcoming Changes

- Added opt-in mK Color Preview decorations for hex, RGB/RGBA, and HSL/HSLA
  colors in any text file, plus an option to enable previews automatically for
  opened files.
- Added default-on enable settings for non-theme editor and workflow features so
  Indent Rainbow, Better Comments, Error Lens, Path Intellisense, Explorer
  layout, Commit Message Editor, and Codex commit generation can be disabled
  independently.
- Removed generated webview bundles from version control; build, check, and
  package workflows now regenerate them when needed.
